Enums in Python

Published on January 1, 0001 in python core

On 12th May 2013, Guido accepted the Python Enhancement Proposal 435 which suggests adding an Enum type to the Python standard library. The news caused a lot of commotion in the Python community and generated (mostly) healthy discussions on Hacker News. In this post I’ll give an introduction to what enums are and talk about how you can use them in your programs. Enumerated Types According to Wikipedia, an enumerated type is a data type consisting of a set of named values called elements, members or enumerators of the type.
