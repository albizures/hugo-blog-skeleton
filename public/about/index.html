This website would not have been possible without the open-source contributions of numerous programmers. It is written in Vim and compiled into HTML using Hugo - a static site generator in Go.

For the design, it uses the excellent minimalistic CSS framework called Skeleton. Beautiful icons on the site are courtesy of the team behind the awesome Ionic Framework.

For typography, it uses the Source Serif Pro open sourced by Adobe. Sans-serif goodness is provided by Raleway.

Lastly, Git is used to publish on Github, where all the code for this website is freely available.
